# Науковий Барометр / Scientists Barometer

Сайт-дашборд інтерсуб'єктивних **індексів якості наукової діяльності в Україні** —
за аналогією з Індексом споживчих настроїв / PMI, але для стану науки.

Дані формуються самими науковцями (опитування) і подаються у двох виглядах: таблиці та графіки.
Перша хвиля — пілотне дослідження N=200 (2026).

## Стек

- **Astro** (статична генерація, zero-JS за замовчуванням) + Observable Plot (графіки — далі)
- Дані: `src/data/wave-YYYY.json` — versioned контракт, генерується з перевірених розрахунків
  (репозиторій аналізу — приватний; сюди потрапляють **лише агрегати**, ніколи мікродані)
- Хостинг: Cloudflare Pages

## Розробка

```sh
npm install
npm run dev      # локальний сервер
npm run build    # статична збірка → dist/
```

## Дані

`src/data/wave-2026.json` (`schemaVersion: 1`) містить: метадані хвилі, 24 індекси з 95% довірчими
інтервалами, не-індексні показники (%), надійність (Cronbach α, Fleiss κ, Spearman), частки робочого
часу та репрезентативність (χ² проти Держстату). Усі числа звірені з дослідженням «Report 2026».

> не потрапляють.
